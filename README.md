## Game Categories

FGL organizes its games into a small number of carefully curated genres.
Each category emphasizes thoughtful gameplay, skill, or relaxed sessions
rather than content aimed at young children.

- <a href="https://fgl.com/tags/match-3/">Match-3 puzzle games</a> built for relaxed, strategic play
- <a href="https://fgl.com/tags/puzzle/">Puzzle and logic games</a> focused on problem-solving
- <a href="https://fgl.com/tags/arcade/">Skill-based arcade games</a> including racing and tower defense

More detailed descriptions of each genre can be found in the `/games` folder.

## About FGL

FGL curates high-quality HTML5 games that work on desktop and mobile devices.
The platform is designed for adult players looking for quick, accessible
gaming sessions without downloads.
